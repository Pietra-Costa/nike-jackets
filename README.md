# Nike Jackets Landing Page

Este projeto é uma landing page responsiva para jaquetas Nike, com carrossel animado usando Swiper.js, menu mobile, e design moderno.

## Funcionalidades

- Carrossel de jaquetas com transição fade, parallax e autoplay
- Paginação interativa do Swiper logo abaixo das imagens
- Menu responsivo com animação de abrir/fechar
- Layout adaptável para desktop e mobile
- Estilo visual com gradientes e fontes modernas

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- [Swiper.js](https://swiperjs.com/) para o carrossel

## Estrutura do Projeto

```
├── index.html         # Página principal
├── styles.css         # Estilos customizados
├── script.js          # Scripts JS para menu e Swiper
└── assets/            # Imagens e logos
```

## Como rodar o projeto

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Pronto! O site estará funcionando localmente

## Personalização

- Para trocar as imagens das jaquetas, substitua os arquivos em `assets/`
- Para alterar cores ou fontes, edite as variáveis em `styles.css`
